## 전략

| 식별자 | UI 이름 | 분배 규칙 |
| --- | --- | --- |
| `even` | 골고루 섞기 | 성별 버킷을 나이 오름차순으로 처리한다. 정확히 같은 나이 안에서는 Fisher-Yates 셔플을 적용하고, 나이가 가까운 버킷부터 snake-direction 슬롯에 분배한다. 각 슬롯 행의 그룹 순서도 셔플해 동일 명단의 결과가 매번 달라지게 한다. |
| `age_similar` | 나이 비슷한 사람끼리 | 전체 명단을 나이 오름차순으로 정렬한다. 정확히 같은 나이 안에서만 셔플한 뒤, 지정된 각 그룹 정원만큼 순차 배정한다. |
| `gender_separated` | 같은 성별끼리 | 남성 우선과 여성 우선 배치를 각각 계산한다. 혼합 그룹 수가 적은 결과를 선택하고, 동률이면 혼합 그룹의 소수 성별 인원이 적은 결과를 선택한다. 따라서 성별은 가능한 한 같은 그룹으로 묶이고, 정원 제약상 필요한 경우에만 혼합된다. |

## 결과 저장

- 새 GroupResult JSON은 `groups`와 `strategy`를 함께 저장한다.
- 기존 GroupResult JSON에 전략 값이 없으면 `even`으로 해석한다.
- 드래그 앤 드롭으로 결과를 수정할 때도 생성 당시의 전략 값을 유지한다.

## 검증 기준

1. UI에서 세 전략 중 하나를 선택할 수 있고 기본값은 `even`이다.
2. 각 전략은 지정된 그룹 정원을 정확히 충족한다.
3. `even` 전략은 같은 명단으로 반복 실행 시 다른 그룹 조합을 만들 수 있다.
4. 결과 생성과 드래그 앤 드롭 저장 뒤에도 전략 값이 GroupResult JSON에 남는다.
