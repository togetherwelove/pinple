- `../인원 리스트 자동 그룹화 서비스 시스템 프롬프트 및 PRD V2.pdf`
- 2026-07-12 사용자 레이아웃 갱신 요청

## 적용 범위

- 대시보드 전용 래퍼를 `flex h-screen overflow-hidden`으로 구성한다.
- 전역 `layout.tsx`는 로그인과 랜딩 페이지에도 적용되므로 변경하지 않는다.
- 데스크톱 사이드바는 `h-full shrink-0 flex-col overflow-hidden`으로 높이와 너비를 고정한다.
- 사이드바의 명단 목록만 `flex-1 overflow-y-auto`로 스크롤하며 계정 영역은 `mt-auto`로 하단에 남긴다.
- 메인 콘텐츠 열은 `flex-1 min-w-0 overflow-y-auto`로 독립 스크롤한다.
- 모바일은 기존의 상단 명단 탐색을 유지하고, 해당 콘텐츠 열만 스크롤한다.

## 검증 기준

1. 데스크톱에서 명단 결과를 스크롤해도 계정 및 로그아웃 영역이 화면 하단에 남아 있다.
2. 사이드바 명단이 길어져도 사이드바의 헤더와 계정 영역은 고정된다.
3. 메인 콘텐츠만 세로로 스크롤되며, 페이지 전체 스크롤바에 의존하지 않는다.
